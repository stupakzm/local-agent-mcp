Create `src/index.ts` — the MCP server entry point with a single stub tool.

Per D-18, D-19, D-20, D-21, D-22, and requirements MCP-01, MCP-03, MCP-04:

```typescript
import { McpServer } from "@modelcontextprotocol/sdk/server/mcp.js";
import { StdioServerTransport } from "@modelcontextprotocol/sdk/server/stdio.js";
import { z } from "zod";

const server = new McpServer({
  name: "local-agent-mcp",
  version: "0.1.0",
});

server.registerTool(
  "run_local_agent",
  {
    description: "Run a task using a local Ollama model. The agent can read files, write files, list directories, and execute shell commands to complete the task.",
    inputSchema: {
      prompt: z.string().describe("The task or question for the local agent"),
      model: z.string().optional().describe("Ollama model name (default: qwen2.5-coder:7b)"),
    },
  },
  async ({ prompt, model }) => {
    try {
      // Stub implementation for Phase 1
      // Phase 2 will replace this with the actual agent loop
      return {
        content: [{ type: "text" as const, text: "not implemented" }],
      };
    } catch (err) {
      return {
        content: [{ type: "text" as const, text: `Error: ${err instanceof Error ? err.message : String(err)}` }],
        isError: true,
      };
    }
  }
);

async function main(): Promise<void> {
  const transport = new StdioServerTransport();
  await server.connect(transport);
  console.error("local-agent-mcp running on stdio");
}

main().catch((err: unknown) => {
  console.error("Fatal:", err);
  process.exit(1);
});
```

CRITICAL RULES:
1. Do NOT use `console.log()` anywhere — only `console.error()`. This prevents stdio transport corruption (MCP-04, D-20).
2. Do NOT wrap `inputSchema` in `z.object()` — pass the plain object of Zod field definitions. The SDK wraps it internally.
3. Do NOT use `server.tool()` — it is deprecated. Use `server.registerTool()` (D-18).
4. The `prompt` and `model` parameters in the handler callback are ALREADY destructured from the first argument. The SDK calls the handler with `(args, extra)`.
5. The try/catch MUST wrap all handler logic so exceptions never escape to the protocol layer (MCP-03, D-22).
6. Use `as const` on the `type: "text"` to satisfy strict TypeScript typing.
